# Пантеон Олимпа — Последний Миф

Галерея-карусель с энкаунтерами арены в стиле греческой мифологии для D&D 2024.

**Живой сайт:** `https://<твой-ник>.github.io/<название-репо>/`

## Порядок карусели

1. «Ревность Геры» — Постоянный энкаунтер · Гера
2. «Нить Ариадны» — Постоянный энкаунтер · Афина
3. «Прилив Леты» — Временный энкаунтер · Аид
4. «Гнев Земли» — Временный энкаунтер · Гефест
5. «Голос Лабиринта» — Временный энкаунтер · Гермес

Отдельно внизу — эпический энкаунтер **«Суд Олимпа»**.

## Как заменить карточку

Просто загрузи новый PNG с тем же именем в `assets/img/` — всё обновится автоматически.

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub
2. Загрузи все файлы (сохраняя структуру папок)
3. **Settings → Pages → Source: main / (root) → Save**
4. Через ~1 мин сайт живёт на `https://<ник>.github.io/<репо>/`

### Через Git

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/<ник>/<репо>.git
git push -u origin main
```
