# НРИ Арена — Последний Миф

Интерактивная галерея энкаунтеров арены в стиле греческой мифологии. Шесть богов — Гера, Афина, Аид, Гефест, Гермес — и финальный Суд Олимпа.

## Содержимое

```
arena_encounters/
├── index.html          — главная страница с каруселью
├── images/             — иллюстрации энкаунтеров
│   ├── hera.png        — Ревность Геры (постоянный)
│   ├── athena.png      — Нить Ариадны (постоянный)
│   ├── hades.png       — Прилив Леты (временный)
│   ├── hephaestus.png  — Гнев Земли (временный)
│   ├── hermes.png      — Голос Лабиринта (временный)
│   └── olympus.png     — Суд Олимпа (эпический)
├── .nojekyll           — отключение Jekyll-обработки
└── README.md
```

## Функции

- Карусель с навигацией стрелками, точками и свайпами
- Клавиатура: ← → для переключения
- Текст описания каждого энкаунтера с типом и механикой
- Адаптивная вёрстка (десктоп + мобильный)
- Open Graph метатеги для превью в соцсетях

## Публикация на GitHub Pages

### Вариант 1 — отдельный репозиторий

1. Создай репозиторий на GitHub (например, `arena-encounters`)
2. Залей содержимое папки в корень репозитория:
   ```bash
   cd arena_encounters
   git init
   git add .
   git commit -m "Initial arena encounters gallery"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_НИК/arena-encounters.git
   git push -u origin main
   ```
3. В настройках репозитория: **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Страница будет доступна по адресу: `https://ТВОЙ_НИК.github.io/arena-encounters/`

### Вариант 2 — подпапка существующего репозитория

Если у тебя уже есть репозиторий с GitHub Pages (например, для Рощи Круга), просто закинь папку туда:

```
существующий-репо/
├── index.html         (главная)
└── arena/             ← положи всё сюда
    ├── index.html
    ├── images/
    └── ...
```

Страница будет доступна по адресу: `https://ТВОЙ_НИК.github.io/РЕПО/arena/`

## Редактирование

Всё содержимое карточек (тексты, названия, описания) находится в массиве `data` в `<script>` блоке внутри `index.html`. Формат записи:

```javascript
{
  title: "«Название»",
  sub: "Тип энкаунтера · Бог",
  badge: "perm",      // perm | temp | epic
  body: [
    "Строка описания 1.",
    "Строка описания 2."
  ],
  closing: "Закрывающая фраза курсивом."
}
```
